Local Storage Example

Use this page to save and edit recipes, create a grocery list, and ... Unfortunately it is only using local storage and will not be saved if you close the browser. It will however keep the information if you refresh the page.

There are many ways to store data in the local storage. You can save it as simple data (or a string), an array, an asociative array, or an object.
