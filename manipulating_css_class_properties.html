Manipulating CSS Class Properties

This is a page all about changing CSS using JavaScript. It isn't difficult to do once you understand Document Object Model. Each part the HTML page can be accessed through the tag names, id, or class names.

In order to access the CSS of each of these elements, all you need to do is put .style after you get the element.

Example:

document.getElementById("myIdName").style

If you want to access the attribute in the CSS, you can add a .attribute to the end of .style.

document.getElementById("myIdName").style.color = "red";

Watch as you click here that the heading will change color.


Now you've changed the background. :)
